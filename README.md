# MATA

A voice-controlled text-to-speech software for blind and visually impaired learners (Grade 12 Research).

## Features
- Extracts text from chosen pdf files
- Utilizes Text-to-Speech to play audio version of the file.
- Voice Control to
  - pause
  - play
  - stop
  - rewind
  - fast forward
  - play next, and
  - previous

## Prerequisites
- Python 3.x
- Virtual Environment
